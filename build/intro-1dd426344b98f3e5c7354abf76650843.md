Welcome! This book proposes numerical tools with python implementation for simulating underground constructions, such as tunnels.


## Tutorials

This section will introduce you to the implementation of machine learning surrogate models using [JAX](https://docs.jax.dev/en/latest/index.html) and the creation of synthetic datasets using [FEniCSx](http://fenicsproject.org). 

- **Notebook 1** – Multi-layer perceptron from scratch
- **Notebook 2** – Shallow tunnel surrogate modeling with FEniCSx and JAX+Equinox
